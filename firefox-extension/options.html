🎆 Button Fireworks Settings

URL Configuration

Fireworks only appear on configured websites. Add URLs below to enable fireworks.

Allowed URLs

Use * as wildcard. Examples:
• https://example.com/* - All pages on example.com
• https://example.com/dashboard - Specific page only
• https://beta3.fwd.app/?/predict?networkId=*&analysisId=*&stepIdx=3 - Wildcard query parameters
• *://*.github.com/* - All GitHub subdomains

Quick Examples (click to add):

Button Configuration

🔘 All Buttons
🎯 Custom Selector
Fireworks appear on all button-like elements (button, [role="button"], etc.).

Custom Button Selector

Quick Examples (click to use):

button.btn.primary
button.btn:nth-child(2)
.submit-btn
[data-testid='submit']
button[type='submit']
#submit-button
Enter any valid CSS selector. The extension will trigger fireworks when elements matching this selector are clicked.
Use DevTools (F12) → Inspector → Right-click element → Copy selector to get exact selector.

Animation Duration

⚡ Short
⏱️ Medium
🎬 Long
Short: Quick celebrations (2x faster fade)
Medium: Balanced timing (default)
Long: Extended celebration (2x slower fade)
